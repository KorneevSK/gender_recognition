1. Для начала необходимо извлечь признаки из аудиофайлов
2. Файлы с извлечёнными признаками сохраняются в отлельную папку
3. Имена файлов с признаками сохраняются с соответствующими метками пола в отдельный csv файл
4. Файл считывается и с помощью функции load_data возвращает вектор объектов и вектор ответов
5. Создаётся модель нейронной сети
6. Происходит обучение модели и её валидация для установления точности модели
7. Модель тестируется на пользовательских данных
